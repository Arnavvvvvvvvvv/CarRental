# 🚗 Car Rental Platform

A full-stack car rental web application where users can browse, book cars, and owners can list and manage their vehicles.

---

## 📌 Features

### 👤 User Features
- Browse available cars
- Search by location and date
- Real-time availability check
- Book cars
- View booking history

---

### 🧑‍💼 Owner Features
- Become an owner (role-based system)
- Add cars with images
- Manage cars (availability, delete)
- View and manage bookings
- Dashboard with stats (revenue, bookings)

---

### 🔐 Authentication
- JWT-based authentication
- Role-based access (User / Owner)
- Protected routes (frontend + backend)

---

## 🧠 Core Functionalities

- 📅 Date-based availability checking
- 🔄 Booking overlap logic
- 🖼️ Image upload using ImageKit
- 🗑️ Soft delete for cars (data consistency)
- ⚡ Global state using React Context
- 🎬 UI animations with Framer Motion

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

### Deployment
- Frontend: Vercel  
- Backend: Vercel (Serverless Functions)  
- Database: MongoDB Atlas  
- Image Storage and optimisation: ImageKit
